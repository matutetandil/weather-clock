## Detailed Description

```
Weather Clock transforms your new tab into a comprehensive weather dashboard with real-time disaster monitoring.

WEATHER AT A GLANCE
- Current temperature, feels-like, humidity, and wind conditions
- Beautiful analog clock with live weather icon
- Sunrise and sunset times with moon phase display
- Air Quality Index (AQI) with color-coded scale
- Choose from 15+ weather forecast models (GFS, ECMWF, MeteoFrance, JMA, and more)

HOURLY & EXTENDED FORECASTS
- 12-hour forecast with animated weather icons
- Sunrise/sunset markers integrated in the hourly timeline
- 16-day extended forecast in a compact grid
- 3D rotating cube interface: swipe between Weather, Tides, and Forecast views

TIDAL INFORMATION
- Today's high and low tides with precise times
- Visual tide bars showing relative heights
- Powered by Stormglass (free API key required for tide data)

MULTI-CITY SUPPORT
- GPS auto-detection for your current location
- Save up to 4 additional cities
- Carousel navigation with swipe gestures, arrows, or dots
- Per-city alert toggles â€” choose which cities get disaster notifications

REAL-TIME DISASTER ALERTS
- Earthquakes worldwide (USGS, GeoNet)
- Severe weather alerts for USA (NWS), Canada (NAAD), Europe (MeteoAlarm), Argentina (SMN), Brazil (INMET), Chile (MeteoChile), and New Zealand (MetService)
- Hurricane and tropical storm tracking (NOAA NHC)
- Chrome desktop notifications for nearby critical events
- Smart alert levels based on actual local impact, not arbitrary thresholds
- Direct links to official alert sources for more information

THEMES & CUSTOMIZATION
- Multiple color themes (Dark, Light, Cream, and more)
- 12-hour or 24-hour time format
- Configurable weather model preference
- Solunar/fishing activity scores

PRIVACY FIRST
- No data collection, no analytics, no tracking
- All preferences stored locally on your device
- Open source: https://github.com/matutetandil/weather-clock

Weather data provided by Open-Meteo. Tide data by Stormglass. Weather icons by Meteocons.
```
